# **App Name**: Niguelito Hub

## Core Features:

- Interactive Hero Section: Displays a welcoming message and a prominent call-to-action button to explore the game collection.
- Game Directory Display: Presents clickable tiles or cards for available games: Math Trivia, Income Simulator, Tetris, Casino Simulator, and Impostor.
- Streamlined Navigation Menu: Provides a simple menu for easy access to different sections of the website or quick links.
- Game Launch Portal: Directs users to the selected game application upon clicking its respective tile or card.
- AI Trivia Question Tool: Utilizes generative AI to create unique and engaging math trivia questions on demand for the Math Trivia game.

## Style Guidelines:

- A vibrant, tech-inspired neon blue (#3D99F5), evoking focus and a futuristic digital feel. (HSL: 210, 90%, 60%)
- A deeply desaturated, subtly blue-tinted dark gray (#161A1D), providing a sleek, immersive dark mode environment. (HSL: 210, 15%, 10%)
- A bright, energetic neon green (#3DF599), offering visual contrast and an engaging dynamic against the primary blue and dark background. (HSL: 150, 90%, 60%)
- 'Space Grotesk' (sans-serif), for a modern, techy, and engaging aesthetic suitable for a gaming portal.
- Utilize modern, geometric, and subtle neon-outlined icons that complement the overall dark and vibrant aesthetic.
- Clean and spacious layout, emphasizing content with a prominent hero section and a grid arrangement for game tiles, ensuring easy readability and navigation.
- Implement subtle hover effects on game tiles and navigation links, with a soft glowing effect for calls-to-action to enhance interactivity and a dynamic feel.
